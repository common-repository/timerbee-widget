== Description ==

# Das Timerbee Widget Plugin zur Online-Terminvergabe:

Das Fenster zu Ihrer sicheren und datenschutzkonformen Online-Terminvergabe mit Timerbee.
Der einfachste Weg, um Ihren Patienten Terminvereinbarungen auf Ihrer Webseite zu bieten.
Entdecken Sie mit unseren Screenshots diverse Ausprägungen, abhängig von der Konfiguration Ihres Timerbee-Systems.

## Ihre Vorteile mit der Timerbee Online-Terminvergabe:

- Ereichbarkeit 24/7
- Entlastung Ihrer medizinischen Fachangestellten
- Terminerinnerung per SMS & E-Mail
- Keine Speicherung sensibler Daten in der Cloud
- Für Smartphones & Tablets optimiert
- Einfache Terminanfrage oder regelbasierter Termin? – Timerbee ist vollständig konfigurierbar
Sehen Sie im Reiter „Installation“ wie einfach die Einbindung des Timerbee Widget Plugins auf ihrer Webseite ist. Das Design des Plugins können Sie nach ihren individuellen Wünschen anpassen.

== Installation ==

Lade das Timerbee Widget Plugin in deinen WordPress hoch, aktiviere es und trage dann für jedes Portal deine Base-Url, Shortname und API-Key ein.
1, 2, 3: Fertig!

== Frequently Asked Questions ==

= Benötige ich zusätzliche Software? =

Ja, nur Nutzer des Online-Terminmanagements Timerbee können dieses Plugin nutzen. Mehr Informationen erhalten Sie unter: https://www.timerbee.de

= Ist das Timerbee Widget Plugin kostenlos?  =

Das Plugins selber ist kostenlos. Um die Funktionen des Plugins zu nutzen, ist eine kostenpflichtige Lizenz des Timerbee Online-Terminmanagements notwendig. Mehr dazu unter:  www.timerbee.de

= Wie lege ich fest welche Behandlungen angezeigt werden? =

Welche Behandlungen online angezeigt werden, können Sie in der Timerbee-Administration individuell konfigurieren.

= Wie können Patienten Termine vereinbaren? =

Patienten werden über das Plugin zu Ihrem Timerbee Online-Terminmanagement weitergeleitet. Hier kann der Termin nach Eingabe der erforderlichen Informationen gebucht oder angefragt werden.
